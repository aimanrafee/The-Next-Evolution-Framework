## 3. PROSEDUR PELANCARAN OFFLINE
Ikuti langkah-langkah ini untuk menjalankan sistem tanpa grid internet:

### Langkah 1: Persediaan Fail
Pastikan semua fail berikut berada dalam satu direktori tunggal:
1. `index.html`
2. `style.css`
3. `script.js`
4. `main.js`
5. `test_integrity.js`

### Langkah 2: Pelancaran (Cold Boot)
* Putuskan sambungan internet peranti anda (untuk simulasi kedaulatan penuh).
* Buka fail `index.html` menggunakan pelayar web moden (Chrome/Brave/Firefox).
* Tekan `F12` untuk melihat **Console** bagi memantau inisialisasi **OSA Engine**.

### Langkah 3: Audit Integriti
* Sistem akan menjalankan `test_integrity.js` secara automatik selepas 3 saat.
* Pastikan semua ujian (Network, Ethical, Vault) menunjukkan status **PASSED ✅**.
